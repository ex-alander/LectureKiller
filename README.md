# LectureKiller

**Превращает нечёткие фотографии лекций в чистые Markdown-конспекты с формулами LaTeX и описанием графиков.**

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## Проблема

Студенты и преподаватели часто сталкиваются с ситуацией, когда качественные материалы отсутствуют, а единственный источник — это **невнятные фотографии с телефона**:
- Текст оборван, графики размыты, формулы «плывут».
- Переписывать конспект вручную — **3–4 часа**.
- Готовиться к экзамену по таким фото — глаза на лоб.

Есть и другая сторона: даже если вы загрузите такой PDF в бесплатный ChatGPT, он **не увидит картинки**, а значит, не сможет объяснить графики или восстановить формулы.

---

## Решение

**LectureKiller** — утилита на Python, которая:
1. **Разрезает PDF** на отдельные изображения страниц.
2. **Отправляет мультимодальный запрос** к LLM (GPT-4o, GPT-5.4, Claude 4.6, Gemini Flash) с инструкцией восстановить:
   - связный текст,
   - формулы в LaTeX,
   - подробное текстовое описание графиков (оси, тренды, ключевые точки).
3. **Сохраняет результат в Markdown** – готовый конспект, понятный человеку и любому LLM.

> **Ключевая фишка**: благодаря детальному описанию графиков, полученный `.md` можно скормить бесплатному DeepSeek, и он ответит на вопросы **так, словно видел исходные слайды**.

## Архитектура

```
PDF-файл
   │
   ▼
[PDF Converter]  →  разбивка на изображения (pillow, pdf2image)
   │
   ▼
[Multimodal LLM Client]  →  параллельные запросы к API (GPT / Claude / Gemini)
   │   (поддержка fallback, retry, rate limiting)
   ▼
[Prompt Builder] →  динамическая инструкция для восстановления контекста
   │
   ▼
[Markdown Generator] →  структурированный вывод с заголовками, списками, LaTeX-блоками
   │
   ▼
.md файл
```

**Технологии**:
- `requests` / `httpx` – асинхронные запросы к API.
- `pdf2image` + `poppler` – конвертация PDF в изображения.
- `pydantic` – валидация ответов моделей.
- `tenacity` – автоматические повторные попытки при ошибках.
- `python-dotenv` – управление ключами.

## Быстрый старт

### Требования
- Python 3.10+
- API-ключ одного из провайдеров (OpenAI / Anthropic / Google AI Studio / ProxyAPI)

### Установка

```bash
git clone https://github.com/ex-alander/LectureKiller.git
cd LectureKiller
python -m venv venv
source venv/bin/activate      # или venv\Scripts\activate на Windows
pip install -r requirements.txt
```

### Настройка

```bash
cp .env.example .env
```

Отредактируйте `.env`:
```ini
API_KEY=ваш_ключ
MODEL_NAME=gpt-4o   # или gpt-5.4, claude-3.5-sonnet, gemini-1.5-flash
```

### Запуск

1. Положите PDF-файлы с фотографиями в папку `input_pdfs/`
2. Выполните:
   ```bash
   python run.py
   ```
3. Результат появится в `output_markdown/` – каждый PDF превратится в одноимённый `.md` файл.

## Пример работы

### Вход (что вы видите)
- Размытая фотография слайда с графиком `v(t)` и формулой `a = dv/dt`.

### Выход (что вы получаете)

```markdown
# Кинематика материальной точки

## Формулы

Скорость: $$v = \frac{ds}{dt}$$
Ускорение: $$a = \frac{dv}{dt} = \frac{d^2s}{dt^2}$$

---

**График 1:** Зависимость скорости от времени.
- Тип: линейный.
- Ось X: время (с), от 0 до 4.
- Ось Y: скорость (м/с), от 0 до 8.
- Уравнение: $v_x(t) = t + 2$ (равноускоренное движение, $a = 1\ \text{м/с}^2$).
```

## FAQ

**Q: Что делать, если на фото почти ничего не видно?**  
A: Модель реконструирует недостающее по контексту всей лекции. Благодаря единому запросу для всех страниц, она «понимает» логику курса и логично дополняет пробелы.

**Q: Распознаются ли действительно сложные графики?**  
A: Да. LLM выдают описание типа графика, осей, ключевых точек и формул аппроксимации. Например: *«экспоненциальный рост, асимптота y=10, точка перегиба при t=5»*.

**Q: Нужно ли платить за ChatGPT Plus?**  
A: Нет. Вы работаете напрямую через API, оплачивая только потреблённые токены.

**Q: А если я не программист?**  
A: Достаточно выполнить 5 шагов из раздела «Быстрый старт» – установить Python, скопировать команды, запустить. Инструкции максимально подробны.
